# Quiz App

A minimal Vite + React quiz app with responsive cross-platform support.

Get started:

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

What I implemented:
- Platform selection for `Web`, `Desktop`, and `Mobile`
- `Load Quiz` flow that loads sample quiz data
- Answer selection and one-by-one question progression
- Final quiz results screen showing score
- Final quiz results screen showing score and percentage, with a Restart button
- Responsive UI for mobile and desktop browsers
- Basic web manifest for installable web app support
